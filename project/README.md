## Laravel Rest Api

Repository ini berisi pembelajaran dasar dalam membangun REST API menggunakan Laravel. Proyek ini mencakup implementasi berbagai HTTP method yang umum digunakan dalam pengembangan API.

## Teknologi yang Digunakan

- PHP
- Laravel
- Postman
- DBeaver

## Fitur

API ini mengimplementasikan beberapa method utama, yaitu:

- **POST**: Menambahkan data baru ke dalam sistem
- **PUT**: Memperbarui data yang sudah ada
- **GET (Show)**: Menampilkan detail data tertentu
- **DELETE (Destroy)**: Menghapus data dari sistem
